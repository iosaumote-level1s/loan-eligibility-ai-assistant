# Loan Eligibility AI Assistant

AI-powered BFSI application using Streamlit + Gemini AI.

## Features
- Loan eligibility calculation
- Risk profiling
- AI banking recommendations
- CSV upload support
- Downloadable reports

## Tech Stack
- Python
- Streamlit
- Pandas
- Google Gemini AI

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
